# Tactical Systems Landing Page

Single-page React/Vite website with a tactical defense-tech visual language. The page is prepared as a client-ready presentation build: responsive layout, polished sections, clean TypeScript build, and typed local demo providers for menu, equipment posts, and contact form behavior.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Current Scope

- Home page with hero, capabilities, services, equipment inventory, aviation specifications, logistics, partners, contact, and footer sections.
- Responsive desktop/mobile layout.
- Client-side demo form validation and success state.
- Local placeholder content providers for menu and equipment records.
